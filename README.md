# beam_search
beam_search 算法python 版本，便于在pytorch中进行更改，优点是可以使用batch_size=1，对于复杂架构的input格式不用repeat数据  
# 随后会增加一个pytorch版本的：  
该写法的优点在于，整体的batch_size都是1，对于复杂的input数据格式类型做beam_search的时候不用去repeat复杂的数据结构。
